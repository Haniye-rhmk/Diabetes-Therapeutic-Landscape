# Data

This folder contains all datasets used in this project.

## Folder Structure

### raw/
Original data obtained from official sources.

Examples:
- Iran Drug Statistics (1400–1404)
- WHO ATC/DDD Index

### processed/
Cleaned and formatted datasets used for analysis.

## Data Sources

- WHO Collaborating Centre for Drug Statistics Methodology
- Iranian National Drug Statistics (Public BI)

## Notes

Raw data should never be modified directly. Data cleaning should be performed using scripts, and the outputs should be stored in the `processed` folder.
