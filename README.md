# Diabetes-Therapeutic-Landscape

Drug utilization analysis of glucose-lowering therapies using the WHO ATC/DDD methodology in Iran.
---

## Research Question

How has the utilization pattern of glucose-lowering therapies in Iran changed between 1400 and 1404?

---

## Objectives

- Identify glucose-lowering drugs
- Retrieve ATC codes
- Retrieve DDD values
- Analyze drug utilization trends
- Compare utilization with clinical guidelines
